## Getting Started

1. **Clone this repository:**
    ```sh
    git clone <repository-url>
    ```

2. **Install dependencies:**
    ```sh
    npm install
    ```

3. **Start the database:**
    ```sh
    docker-compose up
    ```

4. **Start the project:**
    ```sh
    npm start
    ```

5. **Run tests:**
    ```sh
    npm test
    ```
